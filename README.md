# 简历自动填充助手

一键填充简历信息到招聘网站表单的 Chrome 浏览器插件。

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Manifest](https://img.shields.io/badge/manifest-v3-green)
![License](https://img.shields.io/badge/license-MIT-orange)

## 为什么需要它？

海投简历时，每个招聘网站都要重新填一遍姓名、邮箱、学校、实习经历……这个插件让你**填一次，到处用**。

## 功能

- **📂 分类组织**：基本信息、教育背景、实习经历、项目经历、技能语言
- **⚡ 一键填充**：点击侧边栏字段块 → 自动填入网页当前聚焦的输入框
- **✏️ 在线编辑**：切换到编辑模式，直接在面板修改/添加/删除信息，自动保存
- **↕️ 拖拽排序**：编辑模式下字段可自由拖拽排列
- **🔄 一键重置**：恢复默认示例数据
- **🌓 暗色模式**：自动跟随系统主题
- **🧩 兼容主流框架**：支持 React / Vue 受控组件、原生表单、select 下拉框
- **📦 零依赖**：纯原生 JS/CSS/HTML，无构建工具

## 快速开始

### 安装

1. 下载本项目代码
   ```bash
   git clone https://github.com/你的用户名/auto-fill-extension.git
   ```
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 右上角开启「**开发者模式**」
4. 点击「**加载已解压的扩展程序**」
5. 选择项目文件夹
6. 安装完成 👏

### 使用

```
① 打开招聘网站的表单页面
② 点击浏览器工具栏的扩展图标 → 打开侧边栏
③ 在网页表单中点击某个输入框（邮箱/姓名/学校...）
④ 在侧边栏中点击对应的信息块 → 自动填入 ✅
```

### 编辑自己的信息

1. 点击侧边栏顶部的「**✏️ 编辑**」进入编辑模式
2. 点击字段内容即可修改，按 Enter 或失焦保存
3. 拖拽 `⠿` 手柄调整字段顺序
4. 每个分类可 `＋` 添加字段、`✕` 删除字段
5. 底部可「＋ 添加分类」创建新的信息组
6. 点击「🔄」恢复默认示例数据
7. 编辑完毕点击「🔒 填充」回到填充模式

### 🆕 用 AI 助手初始化数据（推荐）

如果你的电脑上有简历文件（`.docx` / `.pdf` / `.txt`），可以让 AI 助手自动读取并填充到插件中：

> 把这个提示词发给你的 AI 助手（Claude Code / Cursor / ChatGPT 等）：
>
> ```
> 我的简历文件在 [你的简历路径]，请读取它的内容，然后修改
> auto-fill-extension/data.js 文件，把 DEFAULT_DATA 里的示例数据
> 替换为我的真实简历信息。保持相同的 JSON 结构和字段分类方式。
> ```

AI 助手会自动：
1. 读取你的简历文件
2. 提取关键信息（基本信息、教育、实习、项目、技能等）
3. 写入 `data.js` 并保持原有分类结构
4. 你在 Chrome 中「重新加载扩展」即可使用

## 常见问题

**Q: 支持哪些网页？**
A: 几乎所有网页表单，包括 React/Vue 框架渲染的表单。

**Q: 数据存在哪里？**
A: 数据存储在浏览器的 `chrome.storage.local` 中，仅本地使用，不上传任何服务器。

**Q: 如何同步到其他电脑？**
A: 编辑好 `data.js` 后重新加载扩展即可。也可以用 Chrome 账号同步扩展数据。

**Q: Chrome 以外的浏览器能用吗？**
A: Edge 可以直接加载。Firefox 需要略微修改 manifest（欢迎 PR）。
